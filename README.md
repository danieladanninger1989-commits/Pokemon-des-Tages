# 🌸 Pokemon-des-Tages

Ein kleines Java-Lernprojekt, das ich während meiner Umschulung zur Fachinformatikerin für Anwendungsentwicklung entwickle.

## 🎯 Ziele

- Java lernen
- Arrays verstehen
- Schleifen üben
- Methoden anwenden
- Eine kleine Benutzeroberfläche erstelle

## 🌸 Projektidee

Jeden Tag wird ein zufälliges Pokémon ausgewählt, das dich begleitet und dir einen kleinen Spruch mit auf den Weg gibt.

Dieses Projekt entsteht Schritt für Schritt als Lernprojekt

## 🚀 Geplante Funktionen

- Zufälliges Pokémon des Tages
- Kleiner Glücksspruch
- Benutzeroberfläche
- Lieblings-Pokémon auswählen
- Weiter kleine Funktionen zum Üben von Java

## 💗Warum dieses Projekt?

Ich lerne Programmieren am besten, wenn ich Projekte entwickle, die mir Freude machen.
Deshalb verbinde ich Java mit meiner Begeisterung für Pokémon.
